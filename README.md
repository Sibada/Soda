#Soda

![py27][py27] ![py35][py35]

微信防撤回工具，依赖itchat模块,just for fun

###Usage

先安装itchat

```
$ pip install itchat
```

然后启动脚本

```
$ python run.py
```
扫描二维码进行登录，就会启动一个网页微信进程，然后可以爽快地获知一切撤回的信息
